Metric Definitions

How To Read Trend Analysis

Trend Analysis shows the four daily indicators behind one selected event or topic. Each chart isolates a single metric so users can see when visibility, engagement, author breadth, or posting momentum changes across the chosen date range.

Metric Definitions

Daily Discussion Volume

The total discussion count captured for the selected cluster on each day. This includes post discussion plus captured comment and sub-comment activity when the source platform provides those fields.

Daily Discussion = Post Discussion + Comments + Sub-comments
Daily Engagement Total

The combined likes, comments, and shares recorded for that cluster on each day. It reflects how strongly audiences are reacting to the content.

Daily Engagement = Likes + Comments + Shares
Daily Unique Authors

The number of distinct authors who posted about that cluster on each day. Higher values indicate broader participation rather than concentration in a few accounts.

Daily Unique Authors = Count of Distinct Authors
Daily Post Volume

The number of posts assigned to that cluster on each day. This is used as the velocity signal because rapid post accumulation usually indicates momentum.

Daily Post Volume = Count of Cluster Posts

How This Connects To Heat

How To Interpret A Spike

A spike means that one indicator increased sharply on that date. When several charts spike together, the event is gaining visibility, interaction, author breadth, and posting momentum at the same time.

How This Relates To Heat Score

Trend charts show raw daily values. The leaderboard heat score uses the same inputs, converts them into weighted factors, and then normalizes the final result onto a 0-100 scale. Use the Heat Analysis explanation panel to review the full scoring logic.
